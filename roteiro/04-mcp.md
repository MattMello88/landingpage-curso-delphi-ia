**Objetivo:** Explicar MCP de forma prática, sem teoria demais

- O que é MCP (Model Context Protocol) em linguagem simples:
  > *"É como dar 'ferramentas extras' para a IA — ela deixa de só falar e começa a agir"*
- Exemplos do que o MCP permite: ler arquivos, executar comandos, acessar banco de dados
- **Instalação prática:**
  - Mostrar o arquivo `mcp.json` (ou `.cursor/mcp.json`)
  - Adicionar um MCP simples (ex: filesystem ou um MCP customizado para Delphi/banco)
- Testar o MCP funcionando no chat
